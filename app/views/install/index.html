Get started

To use the kit you'll need some basic knowledge of HTML, for example what tags and attributes are and how to copy and paste code.

You can learn basic HTML and CSS at mdn, codecademy or other online tutorials.

You'll also need to install some other tools and software.

Choose from the:

  • install guide – for those new to prototyping in code
  • advanced install guide – for those familiar with prototyping in code

NHS England Windows laptops

See the guide for setting up the kit on NHS England Windows laptops.

Updating

If you have already got an NHS prototype but need to upgrade it to the latest version, see the guide to updating the kit.

Switching from the GOV.UK Prototype Kit

If you are new to the NHS prototype kit but are familiar with the GOV.UK Prototype Kit, you can find out how the NHS prototype kit is different from the GOV.UK Prototype Kit.
